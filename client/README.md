# NewsGenie Client (Full UX)

React + Tailwind frontend with:

- Light theme, compact sidebar (S1)
- Home hero layout (Top Headlines)
- Category pages (`/category/:category`) with hero + sidebar list
- Modern auth UI (login/register cards)
- Bookmarks view

## Setup

1. Go to `client/`:

   ```bash
   cd client
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Build production bundle:

   ```bash
   npm run build
   ```

4. The backend in `server/` will serve `../client/build` automatically.
